2. MongoDB Geospatial Search

Store job locations in GeoJSON format:

{
location: {
type: "Point",
coordinates: [longitude, latitude]
}
}

Create a MongoDB 2dsphere index:

jobSchema.index({
location: "2dsphere"
});

Use MongoDB geospatial queries.

Example:

Job.find({
location: {
$near: {
$geometry: {
type: "Point",
coordinates: [lng, lat]
},
$maxDistance: radiusInMeters
}
}
});

Requirements:

* Query only jobs inside selected radius.
* Never fetch all jobs and filter on frontend.
* All filtering must happen inside MongoDB.
